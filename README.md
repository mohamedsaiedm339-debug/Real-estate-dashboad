# 📊 Real Estate Sales Dashboard (Excel)

## 🔍 Overview

This project presents an interactive **Real Estate Sales Dashboard** built entirely in **Microsoft Excel**. It provides insights into property distribution, pricing trends, and sales performance across different locations.

The dashboard is designed to help stakeholders quickly understand market dynamics and make data-driven decisions.

---

## 📁 Dataset

* Total Properties: **150**
* Attributes included:

  * Property Type (Apartment, Office, Villa)
  * Location (Alexandria, Cairo, Giza)
  * Price
  * Availability Status (Sold / Available)

---

## 📌 Key Metrics

* **Total Properties:** 150
* **Average Price:** $2,602,881

---

## 📊 Dashboard Features

### 1. Total Sold vs Available

* Visual comparison using a horizontal bar chart
* Helps track inventory vs completed sales

### 2. Property Distribution by Type

* Donut chart visualization:

  * Apartments: **56%**
  * Offices: **25%**
  * Villas: **19%**

### 3. Sales by Location and Property Type

* Clustered column chart comparing:

  * Alexandria
  * Cairo
  * Giza
* Segmented by property type for deeper insights
